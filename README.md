## How To Use


```bash
# Clone 
$ git clone https://github.com/gportodev/test-coursify.me.git

# Go into
$ cd test-coursify.me

# Install dependencies
$ yarn install

# Run app
$ expo start
```

## Additional notes

In order to run you need to download the [Expo client](https://expo.dev/client).
